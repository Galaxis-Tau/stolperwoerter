# Stolperwörtertest – Form B

Eine webbasierte Version des **Stolperwörtertests** für den Einsatz im Unterricht oder zur Leseförderung.

Die Anwendung zeigt nacheinander Sätze an, in denen jeweils ein Wort nicht dazugehört. Die Nutzerinnen und Nutzer klicken das Stolperwort an. Am Ende wird automatisch eine Auswertung angezeigt.

## Funktionen

- 60 vorbereitete Testsätze
- Zufällige, ausgewogene Reihenfolge der Aufgaben
- 4-Minuten-Timer
- Automatischer Wechsel nach Auswahl eines Wortes
- Möglichkeit zum Überspringen einzelner Sätze
- Sofortige Auswertung am Ende
- Anzeige von:
  - bearbeiteten Sätzen
  - richtigen Antworten
  - falschen oder übersprungenen Aufgaben
  - Trefferquote in Prozent
- Einzelübersicht aller bearbeiteten Sätze
- Neustart des Tests ohne Neuladen der Seite
- Responsives Design für Desktop, Tablet und Smartphone
